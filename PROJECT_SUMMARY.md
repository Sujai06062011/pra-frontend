### 🚀 How to Use After Download

1. **Extract the ZIP file**
   ```bash
   unzip patient-relationship-assistant.zip
   cd code
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   # or
   npm install
   ```

3. **Run development server**
   ```bash
   pnpm run dev
   # or
   npm run dev
   ```

4. **Build for production**
   ```bash
   pnpm run build
   # or
   npm run build
   ```

### 🌐 Deploy to Cloud

**Vercel (Recommended)**
```bash
npm i -g vercel
```

**Netlify**
```bash
npm i -g netlify-cli
```
